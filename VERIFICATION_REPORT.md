## How to Run

```bash
# 1. Start Ollama (required)
ollama serve
ollama pull llama3

# 2. Start Backend (Terminal 1)
cd backend
npm install
node server.js
# → http://localhost:5000

# 3. Start Frontend (Terminal 2)
cd frontend
npm install
npm run dev
# → http://localhost:5174
```
